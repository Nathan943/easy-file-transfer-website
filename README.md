# Easy File Transfer

Easy File Transfer is a small web application I have been building to quickly transfer large files between many devices. Devices can pair using a generated pairing code, and transfers happen through WebSockets. Transfers across networks are still a work in progress, but it currently works across local networks with a local server host.

## Built With

- React
- TypeScript
- Vite
- Node.js
- WebSockets
- IndexedDB
- Bootstrap
